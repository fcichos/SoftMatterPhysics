Gibbs Distribution¶

So far we have considered closed systems and derived and studied the Boltzmann distribution. Now our systems are open the particle number might change. The Boltzmann distribution is therefore not anymore sufficient.

\begin{equation} \frac{p(E_{s}^{(1)},N_s^{(1)})}{p(E_{s}^{(2)},N_s^{(2)})}=\frac{W_r(E_{tot}-E_s^{(1)},N_{tot}^{(1)})}{W_r(E_{tot}-E_s^{(2)},N_{tot}^{(2)})} \end{equation}

Hereby, the \(E_{s}^{(i)}, N_s^{(i)}\) denote the energy and the number of particles in the system. The \(W_{r}\) indicate the number of available states to the reservoir, when the system is in a specific energy state with a specific number of particles. Similar to the derivation of the Boltzmann distribution we can now assert, that the total number of available states of the system plut reservoir is given by

\begin{equation} W_{tot}(E_{tot}-E_{s}^{(1)},N_{tot}-N_{s}^{(1)})=1\times W_{r}(E_{tot}-E_{s}^{(1)},N_{tot}-N_{s}^{(1)}) \end{equation}

and using again \(S=k_B \ln(W)\) or \(W=\exp(S/k_B)\) we find

\begin{equation} \frac{W_r^{(1)}}{W_r^{(2)}}=\frac{\exp(S_r^{(1)}/k_B )}{\exp(S_r^{(2)}/k_B)} \end{equation}

We can now use a first order expansion again to get

\begin{equation} S_{r}(E_{tot}-E_{s},N_{tot}-N_s)\approx S_{r}(E_{tot},N_{tot})-\frac{\partial S_{r}}{\partial E}E_{s}-\frac{\partial S_{r}}{\partial N}E_{N_s} \end{equation}

The partial derivatives can be converted with the help of the known thermodynamic relations

\begin{equation} \frac{\partial S_{r}}{\partial E}\bigg|_{V,N} =\frac{1}{T} \end{equation}

and

\begin{equation} \frac{\partial S_{r}}{\partial N}\bigg|_{V,E}=-\frac{\mu}{T} \end{equation}

which finally yields the Gibbs distribution:

Gibbs-distribution

\begin{equation} p(E_{s}^{(1)},N_s^{(1)})=\frac{1}{Z}\exp\left ( -\frac{E_{s}^{(1)}-\mu N_{s}^{(1)} }{k_B T} \right ) \end{equation}

with the grand partition function

\begin{equation} Z=\sum_{i}\exp \left ( -\frac{E_{s}^{(i)}-\mu N_{s}^{(i)}}{k_B T}\right ) \end{equation}

Ligand Binding

As an example, we have a look at the binding of a ligand (green) to a receptor, which is indicated in the figure below.

The binding is reported by the variable \(\sigma\), which can be either 0 or 1. The binding energy is therefore

\begin{equation} E=\sigma \varepsilon_b \end{equation}

with \(\varepsilon_b<0\), since the ligand wants to bind. The grand partition function can therefore be written as

\begin{equation} Z=\sum_{\rm states}=\exp\left ( -\beta (E_{\rm state}- \mu N_{\rm state}) \right ) \end{equation}

in which the chemical potential \(\mu\) reflects the contact with the particle reservoir.

\begin{equation} Z=\sum_{\sigma=0}^{1}=\exp\left ( -\beta (\varepsilon_b \sigma- \mu \sigma) \right )=1+\exp(-\beta (\varepsilon_b -\mu )) \end{equation}

From that we can calculate the mean bounda ligands to be

\begin{equation} \langle N \rangle =\frac{\exp(-\beta (\varepsilon_b -\mu ))}{1+\exp(-\beta (\varepsilon_b -\mu ))} \end{equation}

Using

\begin{equation} \mu = \mu_0 + k_B T \ln \left ( \frac{c}{c_0}\right ) \end{equation}

we can finally write

\begin{equation} \langle N \rangle =\frac{c/c_0 \exp(-\beta \Delta \varepsilon_b)}{1+\frac{c}{c_0}\exp(-\beta \Delta \varepsilon_b)} \end{equation}

with \(\Delta \varepsilon=\varepsilon_b-\mu_0\), which is the energy freed when taking the ligand from the solution and placing it at the acceptor.
